# latest-alpha — between-filings discovery log

Output of the `/latest-alpha` skill. Each file consolidates **one company's recent developments** between the lagging 10-Q/10-K cycles — SEC 8-K / 424B5 / 6-K (Tier-1, timely), investor-relations press releases, and reliable news — tier-ranked by impact, ending in a list of **primary-source-verifiable leads**.

**This is a Tier-3/4 discovery log, NOT vault canon.** Same status as `video-intel/`:
- Not in `index.md` / `log.md`; these files do not count for accounting.
- Nothing here is ground truth. Per CLAUDE.md Section 2.2 / Section 4.6, IR / press / news are not eligible to be written onto canonical pages as fact.
- The payoff is the "verify at primary" lead list. Anything that should reach a `wiki/` page goes through the normal **human-gated primary-source ingest** (the two-stop protocol), separately.

File naming: `<RUN_DATE>_<TICKER>_recent-developments.md` (e.g. `2026-06-09_AAOI_recent-developments.md`).
