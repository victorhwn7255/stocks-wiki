# video-intel — YouTube intelligence log (Tier 3/5)

Output of the `/youtube-intel` skill. Each file is a vault-aware analysis of one YouTube video: source-Tier verdict + TL;DR + named entities + claims + vault cross-reference + primary-source verification leads, with the cleaned transcript appended.

**These are NOT vault canon.** YouTube is Tier 3 (independent analysis — "cite, don't treat as fact") at best, Tier 5 (social — "never cited") at worst, per CLAUDE.md Section 2.2. Nothing here is a primary source. The notes exist to turn watched videos into a greppable feed of *questions and ingest leads*; anything that reaches a wiki page must first be verified against primary filings/calls through the normal human-gated ingest.

Not in `index.md` / `log.md`; these files do not count for accounting (same convention as `refresh_log.md` and `conventions-ledger.md`).

Filename convention: `<upload-date>_<channel-slug>_<title-slug>.md`.
