# Self-research topics — the vault's living deep-research agenda

The priority-ranked research questions "Stocks Wiki" works through. **Tue–Sat at 02:30 SGT**, the
automation picks the **top Pending** item, runs `/deep-research`, saves the report to
`raw/research/self-research/`, and moves the item to **Researched** (so it never repeats and the
list rotates day by day).

**Lifecycle (part of self-learning):**
- **Pending** — Vic-curated, ranked. The automation researches the top item each research-day.
- **Researched** — auto-moved here with the date + report path once done.
- **Proposed (by automation)** — candidates the daily scan surfaces from the vault's own open
  questions / connection-finder clusters / coverage gaps. Vic promotes the good ones into Pending.

**Rules:**
- Entries must be **well-scoped questions** (the unattended run can't ask clarifying questions).
- Reports are **Tier-3 discovery material**, human-gated to canon — a report does NOT become a
  theme page automatically (that stays the §3.13 two-stop). This file + `self-research/` live in
  `raw/research/` (not `wiki/`), so it's within the discovery-only contract.
